## Тренировочные и тестовые данные
Исходный dataset представляет из себя трехмерную матрицу размера 512x512x300, как результат обработки видеоряда (512x512 - размер кадра, 300 - количество кадров). После чего происходит формирование выборки относительно каждого из пикселей с учетом хронологии его изменений. В результате чего получается общая выборка размерности 262144x300 (где 300 - количество признаков, 262144 - объем выборки).
Тренировочная и тестовая выборки формируются случайным образом из общей в соотношении `0.33 : 0.67`.

## Разработанные скрипты
* `autoencoder_conf.py` - скрипт, который задает конфигурацию автокодировщика (4 различных варианта конфигурации).
* `load_data2.py` – скрипт для предобработки входных данных из лабораторной работы №2.
* `load_data3.py` – скрипт для предобработки входных данных из лабораторной работы №3.
* `pretrain.py` – решает задачу построения автокодировщика для заданной конфигурации нейронной сети.
* `pretrain_stack.py` – решает задачу построения стека автокодировщиков для заданной конфигурации нейронной сети.
* `train.py` – скрипт, который осуществляет обучение заданной конфигурации нейронной сети.
* `save_data.py` – скрипт для сохранения полученной конфигурации нейронной сети в файл в `.npy` формате.
*	`lab4_conf*.py` - скрипт для запуска и проведения полного цикла рассчетов с использованием автокодировщика для заданной конфигурации.
*	`lab4_conf*_stack.py` - скрипт для запуска и проведения полного цикла рассчетов с использованием стека автокодировщиков для заданной конфигурации.
